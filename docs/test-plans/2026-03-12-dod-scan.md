## End-to-End: Full Pipeline Integrity

1. Start with fresh database: `dod-scan init-db`
2. Run `dod-scan run-all --backfill 0` (today's page only)
3. Verify exit code 0 and log file at `logs/dod_scan.log` contains stage start/complete entries
4. Spot-check SQLite database:
   - `pages` table has rows with non-empty `raw_html`
   - `contracts` table has plausible `company_name`, `dollar_amount` > 0
   - `classifications` table has `is_procurement` in {0, 1}, `confidence` between 0-1
   - `contract_locations` has `latitude` 24-50, `longitude` -125 to -66 (continental US)
5. Open KML in Google Earth; verify placemarks with coloured icons and popups
6. Open HTML map in browser; verify pins, popups, and filters
